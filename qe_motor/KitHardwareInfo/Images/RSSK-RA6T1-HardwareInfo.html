Motor Control Evaluation System for RA Family - RA6T1 Group

RTK0EMA170S00020BJ

The Motor Control Evaluation System for RA Family - RA6T1 Group is a low-voltage permanent magnet synchronous motor (brushless DC motor) solution that allows anyone to easily evaluate motor control. Start evaluating motor control immediately using the kit hardware and software available for download from the Renesas website.

Product Specification

    Features
    • The CPU board is equipped with the RA6T1 as the motor control MCU.
    • Inverter board for 3-phase BLDC motor (48V/10A rating)
    • On-board debugger for flash programming of RA6T1
    • Supports 1-shunt/3-shunt current sensing
    • Overcurrent detection
    • Supports the motor control development support tool, Renesas Motor Workbench ( Introduction to the use of Motor Workbench (with videos) )
    • Motor control evaluation and debugging can be performed safely with the communication board which isolates PC and the CPU board electrically
    • Supports Hall sensors, encoders, and inductive position sensors
    • Included items
      • RA6T1 CPU board
      • Permanent magnet synchronous motors
      • Accessories (cables, standoffs, etc.)
